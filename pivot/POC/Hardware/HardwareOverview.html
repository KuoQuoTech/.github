Hardware Overview¶

System Architecture¶

The proof of concept combines three primary hardware modules:

Module Purpose
Renesas DA14531 Ultra-low-power BLE MCU
LSM6DSOX 6-DOF IMU Motion + gesture sensing
MAX86140 Optical heart rate + SpO2 sensing

Together, these components form a compact wearable sensing platform optimized for smart ring development.

Core MCU + BLE¶

Renesas DA14531 Dev Kit¶

The system uses the DA14531 ultra-low-power Bluetooth Low Energy (BLE) SoC as the primary controller.

Key Responsibilities¶

  • BLE communication
  • Sensor data collection
  • Power management
  • Real-time event processing
  • Mobile app connectivity
  • Gesture/event detection

Key Advantages¶

  • Extremely low power consumption
  • Optimized for wearable applications
  • BLE 5 connectivity
  • Compact footprint for ring-sized devices
  • Fast wake-up and sleep transitions
  • Suitable for battery-powered continuous sensing

Motion Sensing¶

LSM6DSOX 6-DOF STEMMA QT / QWIIC¶

The development board integrates the LSM6DSOX 6-axis IMU for motion sensing and gesture detection.

Sensor Features¶

  • 3-axis accelerometer
  • 3-axis gyroscope
  • Embedded machine learning core
  • Motion classification support
  • Step/activity detection
  • Tap and orientation detection

Potential Use Cases¶

  • Gesture control
  • Air gestures
  • Activity tracking
  • Motion-triggered events
  • Sleep movement analysis
  • Wearable interaction experiments

Biometric Sensing¶

MAX86140 Evaluation Module (MAX86140EVSYS)¶

The MAX86140 optical sensor module enables heart rate and pulse oximetry experimentation.

Sensor Features¶

  • Heart rate monitoring
  • SpO2 estimation
  • High sensitivity optical readings
  • Low-power wearable operation
  • Integrated analog front end

Potential Use Cases¶

  • Wellness tracking
  • Fitness monitoring
  • Stress analysis research
  • Real-time biometric streaming
  • AI-assisted health insights
